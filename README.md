# Analyse et prédiction des prix immobiliers

## Description
Ce projet analyse les facteurs influençant les prix des maisons et utilise des
modèles de machine learning pour prédire et classifier les prix immobiliers.

## Contenu du projet
- Nettoyage et analyse exploratoire des données
- Régression linéaire
- Régression logistique
- Courbe ROC et matrice de confusion

## Exécution
Ouvrir le notebook Jupyter pour exécuter l’analyse complète.
